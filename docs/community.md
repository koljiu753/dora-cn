## 投稿 {#contribute}

### 你写教程，我们帮你扩散

只要你在用 Dora 搞机器人或具身智能，哪怕只是一个 demo 记录——发我们 PR 或公众号投稿，我们帮你润色、上首页、推到 GitHub Discussions。

### 投稿三步

1. **Fork 仓库** [koljiu753/dora-cn](https://github.com/koljiu753/dora-cn)
2. **添加你的文章**到 `.vitepress/theme/data/articles.json`
3. **提 PR**，等待审核（通常 2-3 个工作日）

### 投稿模板

```json
{
  "title": "你的文章标题",
  "url": "https://你的文章链接",
  "author": "你的昵称",
  "platform": "CSDN / 知乎 / 个人博客 / ...",
  "date": "2025-11-15",
  "category": "tutorial | community | deep-dive | news",
  "categoryLabel": "教程 | 社区 | 深度 | 资讯",
  "summary": "一句话简介，30 字左右"
}
```

## 社区准则

我们希望这是个**纯粹、友好、高质量的技术交流环境**：

- 互相尊重，对事不对人
- 提问前先尝试自己解决，提问时附上完整上下文（系统、版本、代码、报错）
- 鼓励原创内容，转载注明出处
- 反对营销刷屏、人身攻击、违法内容

违反准则的内容会被删除，多次违规的成员会被移除社区。
